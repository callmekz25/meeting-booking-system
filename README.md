# Meeting Management system

## Project Overview
Meeting Equipment Management is a web application designed to help organizations manage meeting rooms, equipment, and bookings efficiently. The system ensures that meetings are scheduled without conflicts and that equipment inventory is properly tracked and updated.

## Features

### Meeting Booking Management
- Create, update, and cancel meeting bookings.
- Check for room availability and prevent scheduling conflicts.
- Send notifications to attendees and organizers via email upon booking creation or updates.

### Meeting Room Management
- Create, update meeting rooms.
- Assign available equipment to each room.
- Track room capacity and equipment availability.
  
### Equipment Inventory Management
- Track equipment quantity and status in meeting rooms.
- Update equipment inventory via file upload for bulk operations.

### User Management
- Admin provide account for internal users
- Role-based permissions (admin, user).

### Conflict Prevention
- Prevent double booking of rooms and overlapping meetings.

### Notifications
- Email notifications for new bookings, updates, and cancellations.

## Tech Stack
- **Frontend:** React, TypeScript
- **Backend:** .NET Core (C#)
- **Database:** MySQL 
- **Message Queue:** RabbitMQ
- **Cloud Storage:** Cloudinary
